# 队伍分配工具

一个纯静态网页工具，用于维护人员名单、选择参与分队的人员、设置每队人数与队伍数量，并在自动分队后手动调整结果。

## 功能

- 添加、删除人员名单
- 勾选本次参与分队的具体人员
- 设置每队人数和队伍数量
- 支持随机打乱后自动分队
- 支持把人员手动移动到任意队伍
- 使用 MantleDB 公开共享房间保存当前名单和分队结果
- 多人打开同一个页面可共享数据，并自动定时同步最新结果

## 本地预览

直接打开 `index.html` 即可使用，也可以在项目目录启动任意静态文件服务器。

## 在线预览

项目已准备 GitHub Pages 工作流。推送到 GitHub 后，仓库会通过 GitHub Actions 自动发布静态页面。

## 数据说明

当前版本使用未声明的 MantleDB namespace 作为公开共享数据存储，不需要账号或密钥。知道页面地址的人都可以读取和修改共享数据，请不要保存隐私信息。
